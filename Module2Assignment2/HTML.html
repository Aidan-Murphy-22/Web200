HTML as said before is the foundation of a webpage. HTML stands for Hypertext Markup Language and is what is used to create the basic structure of a webpage. It is used primarily to organize and present content on a webpage and also to link one webpage to another. It allows web developers to add text, images, other multimedia, and even interactive elements that are necessary for forms.

The language is comprised of tags that are essential to the formatting of a webpage. These include tags such as head, title, body,h tags, and p. These are all tages that are enclosed in < angle brackets like this >. Other HTML tags include a tags(anchor tags) which create links,img tags that insert images into the webpage, list tags like ol(ordered list), ul(unordered list), and li(list item) to just name a few. These tags are all necessary to create a fuctioning webpage.
